Webhook

GitHub webhook endpoint for automatic repository updates:

POST {{ request.base_url }}webhook/github

Configure this URL in your GitHub repository settings under Webhooks.

Example: Create Repository
curl -X POST {{ request.base_url }}api/repositories \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "my-repo",
    "url": "https://github.com/user/repo.git",
    "branch": "main"
  }'
Example: Sync Repository
curl -X POST {{ request.base_url }}api/repositories/1/sync \
  -H "Authorization: Bearer YOUR_API_KEY"
Response Format

All API responses return JSON with the following structure:

{
  "success": true,
  "data": {...},
  "message": "Operation completed successfully"
}
